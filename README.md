Overall project description
The nebula calculator is a four-function calculator.
 This type calculator performs operations in the order they are entered, without considering operator precedence.
(Does not follow BODMAS)
The Nebula Calculator is a scientific calculator programmed in a language called c

Special keys
The nebula calculator includes other keys such as the inv, M, M+, M-, ANS and the backspace key.
	Use of each key
•	Inv  	This key is used to alternate between functions e.g. ²√ and ³√.
•	M	This key is used to retrieve values from the memory.
•	M+	This key is used to mount a value to the memory.
•	M-	This key is used clear the memory.
•	ANS	This key is used to retrieve the result of the previous input 
•	backspace	Clears a value from the input including the operators. i.e. +, -, etc.
•	AC	This key clears the input display and memory.
•	C	This key clears the equation display.

More functions and keys shall be added in future including the ability to use BODMAS.
